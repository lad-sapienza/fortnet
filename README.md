## About the Project

FortNet focuses on the coastal region of ancient Chaonia (Southern Albania), an area that has been relatively marginal in archaeological studies, which have primarily concentrated on larger urban sites such as Phoinike, Antigonea, and Butrint.

The project aims to:
- Provide a complete census of open and fortified sites in the coastal Chaonia region
- Create detailed topographical documentation using modern surveying technologies and geophysical methods
- Study settlement patterns, fortification networks, and population dynamics from the late Bronze Age to the Medieval period
- Build a shared web-based geodatabase for collaborative research
- Engage with local communities to balance heritage protection with territorial resource exploitation
